# 📚 Library Management System

## 📖 About the Project

The **Library Management System** is a Python console application that allows users to manage books in a library.

This project demonstrates the use of Python dictionaries, conditional statements, loops, and user input to perform basic library operations.

---

## ✨ Features

- ➕ Add a new book
- 📚 View all books
- 🔍 Search for a book
- 📖 Borrow a book
- 🔄 Return a book
- 🗑️ Delete a book
- 🚪 Exit the program

---

## 🛠️ Concepts Used

- Python Dictionaries
- Nested Dictionaries
- Variables
- User Input (`input()`)
- Conditional Statements (`if`, `elif`, `else`)
- Loops (`while`)
- Boolean Values (`True` / `False`)
- Dictionary Operations (`in`, `del`)
- String Formatting
- CRUD Operations (Create, Read, Update, Delete)

---

## ▶️ How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open the project in VS Code or any Python IDE.
4. Run:

```bash
python Library-Management-System.py
```
